# サービスの説明
「Target Management」  
自分の目標を管理・共有し、目標達成をサポートするサービスです。
本サービスを使用することで、以下のメリットがあります。
- 管理  
  - 期日を設定する
  - 目標を細分化する  
- 共有
  - 宣言することで自分を追い込む  
  - 人から応援してもらえる  
  - アドバイス、コメントをもらえる
